# Accounting Ledger Application

## Overview 
**This Accounting Ledger Application can be used to track all financial transactions for a business or personal use. The software is able to take user input and save it to a transactions.csv file.**

Users are able to: 
1. View all transaction entries 
2. Record deposits 
3. Record payments 
4. Run pre-defined reports or preform a custom search to anaylize transaction history

## Installation 
1. Clone the repository
2. Navigate to project directory
3. Compile application (we are using Java)
4. Run the program from the main
